# vim

vim8 vs vim9

## vim 版本

安装不同的 vim 版本：
- vim-minimal/vim-tiny，最小版本，不支持 Python/Lua/Ruby，Ubuntu 默认安装的版本
- vim-basic，基础版本，支持 Python
- vim-nox，完整版本，支持 Python/Lua/Ruby
- vim-gtk3，完整版本 + GUI图形化界面

## 思考

一款编辑器最核心的功能是文件的浏览和编辑：

关于文件浏览，怎么才能看的更好：
1. 文件翻页，跳转，函数查找，高亮；
2. 代码对齐，格式化；括号匹配；
